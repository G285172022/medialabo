「Webコンテンツ作成」の成果

本テーマの学習目的

  • HTML によって Web ページの構造を記述する基本的な方法がわかる
  • CSS によって Web ページの見た目を設定する基本的な方法がわかる
  • JavaScript によって Web ページのインタラクティブな処理を記述する基本的な方法がわかる
  • HTML, CSS, JavaScript の構文リファレンスから，構文の書式や使用方法の情報を入手できる
  • HTML, CSS, JavaScript, 画像・音声・動画ファイルを組み合わせて簡単なWebページが作成できる

プロジェクト課題について

プロジェクト課題では、NHKの番組表の検索を実施する。 表示する内容は、検索欄、検索結果の件数、検索結果の番組情報の一覧である。
検索方法は、チェックボックスでチャンネルを、プルダウンメニューで番組のジャンルを指定するようにした。 検索するチャンネルが一つも選択されないまま検索ボタンが押されたときは、選択するように促すメッセージを表示する。 また、2つのチャンネルが選択されたときも、正しくすべての検索結果が表示される。

プロジェクト課題で作成したページ: project.html

数当てゲーム

この課題では、ランダムに生成される答えの数字を、入力した数字との大小比較の情報をもとに当てるゲームの作成を実施する。
第3回ではJavaScriptを用いて数字の生成と入力した数字と生成した数字の大小の判定処理を実装した。ここでは判定の結果をWebページ上に表示することができないため、結果はコンソールに表示した。
第4回ではDOMプログラミングでHTMLを編集することで、予想回数や予想の数字、第3回で実装した判定処理の結果をWebページ上に表示できるようにした。
第5回では数字の入力欄と回答ボタンを設置して、イベント処理を用いて入力欄に入力された数字と答えの数字との大小比較の処理をボタンで呼び出すという処理を実装した。

数当てゲームのページ: kazuate.html

毎回の実験で作成したページ

第1回 WWWとHTML

今回はWWWの技術要素やWebページの構成、HTMLの基礎的な記述方法を学んだ。 HTMLを用いることで、Webページの構造を記述することができる。
また、次の課題に取り組んだ。演習1-3ではHTMLの基本的な構造を、演習1-4ではHTMLの要素の使い方を学んだ。

演習の成果

  • 演習1-3: HTML文書の作成
  • 演習1-4: HTMLの様々な要素

第2回 CSSによるWebページの装飾

今回はCSSを用いたWebページの装飾の方法について学んだ。 CSSを用いることで、要素の配置を変更したり、背景やテキストの色を変更したり、枠線をつけたりすることなどができる。
また、次の課題に取り組んだ。演習2-3では、CSSを用いたWebページのレイアウトの方法を学んだ。

演習の成果

  • 演習2-3: ウェブブラウザ説明ページの装飾

第3回 JavaScriptの基本

今回はJavaScriptを用いた基本的なWebプログラミングの手法を学んだ。 JavaScriptを用いることで、ページを開いたときやボタンを押したときに、ページ上の記述やデザインを変化させるなど、Webページ上で複雑な機能を実現することができる。
また、次の課題に取り組んだ。演習3-6ではオブジェクトのメンバーを参照する方法を学んだ。

演習の成果

  • 演習3-6: オブジェクトの参照

第4回 DOMとJavaScript

今回はJavaScriptを用いてHTMLやCSSの要素を編集するDOMプログラミングの手法を学んだ。 DOMプログラミングを用いることで、JavaScriptで処理した結果などをWebページ上に表示することができる。
また、次の課題に取り組んだ。演習4-2では要素の追加方法を、 演習4-3では要素内への要素の追加方法を、演習4-4では要素の削除方法を, 演習4-5ではオブジェクトのリスト要素への追加方法を学んだ。

演習の成果

  • 演習4-2~4-5: DOMプログラミング

第5回 イベント処理

今回はイベントが発生した際にプログラムが動作する、イベント処理の実装方法を学んだ。 イベント処理を用いることで、ボタンが押されたり要素のマウスオーバーやクリックがされたりした際などにJavaScriptプログラムを実行することができる。
また、次の課題に取り組んだ。演習5-3ではさまざまなイベントの呼び出されるタイミングを学んだ。
